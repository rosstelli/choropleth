How To Use

The initial display is information of the median age according to each county.

To interact with the graph, click on any county. An orange dot will appear on the county selected. On the right hand side of the figure, the county information will display in the top rectangle. As the mouse moves around the map, a green circle appears in the county that the mouse is currently hovered over. The information for this county will display on the bottom right rectangle. This creates an easy to use comparison between any two counties.

The sliders at the top act as a threshold for minimum and maximum of an unemployment rate to appear, respectively. This means that the user can create any range of values to be observed in the map. Counties that do not fall in this range will turn light-gray.

To select different datasets, simply press a number key between 1 and 7. The information for each will display as such:
  1. Population
  2. Median Age
  3. Percentage of Population Employed
  4. Private nonfarm Establishments
  5. Private nonfarm annual payroll
  6. Births Per 1000 People
  7. Deaths Per 1000 People

Developer Notes

To add a new data set, add the data to a column in the all_data.csv file in the data folder. Name the column appropriately. In the following code add the appropriate line to load data in:
queue()
.defer(d3.json, "json/us.json")
.defer(d3.csv, "data/all_data.csv", function(d) {
temp.data = +d.data;
rateById.set(d.id, temp);
})
.await(ready);
In the scripts/toggle.js file, add a function to var functions that coordinates with the new value added. Copy and paste a previous function as a 'template'.
The last step, add a listener to a key that has not been used yet. For a template copy any if statement (except (key === 82)) in the window.onkeyup function. Make sure to adjust parameters as necessary.
